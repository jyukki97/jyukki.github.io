# 위상 정렬

###### 방향 그래프의 노드에 대해 순서를 매겨 정렬하는 방법

###### 정렬 후에 각 노드에 대해 순서가 위배되지 않아야한다.

###### 사이클이 있다면 불가능하다.





###### 들어오는 간선이 없는 노드를 삭제하면서, 정렬한다.
